How to Apply CSS to HTML

To include an external CSS file in an HTML document, you can use the <link> tag inside the <head> section. For example: 

        <link rel="stylesheet" href="styles.css">

How to Use JavaScript in a Webpage

To include JavaScript in an HTML document, you can link an external JavaScript file using the <script> tag: 

        <script src="script.js"></script>
